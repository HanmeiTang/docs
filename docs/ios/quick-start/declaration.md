通过本章教程，熟悉白板的从创建到结束的声明周期，以及事件回调。

## 流程

1. 在 sdk 后台创建账号，并将 SDK 集成如 iOS 项目中
1. 在 iOS 项目中，创建白板实例，初始化 SDK
1. 通过与 sdk 业务服务器通讯，在 sdk 服务器端，创建房间
1. 获得房间 uuid 与 roomToken，调用 sdk 加入房间方法
1. 实现白板房间状态回调协议
