## 运行环境

sdk 可以运行在任意支持 es5 的现代浏览器中，兼容性支持可以查看 [caniuse-ECMAScript 5](https://caniuse.com/#feat=es5)。  
硬件上，sdk 支持手写板，iPad 等设备。
