## SDK 版本列表

### 2.0.0 正式版 - 2019-06-23

正式版与之前beta 版本，API 基本一致。但是不能与低版本sdk 进行互联。可与 web 2.0.0 正式版，以及 iOS 2.1.0 版本进行互联。

>2019.7 前接入客户，在升级前请联系 SDK 团队，切换服务器配置。
